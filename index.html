Supabase

Checklist de publicacao

  1. Crie um projeto no Supabase e cole a URL e a anon key em config.js.
  2. Rode o SQL do arquivo supabase/schema.sql no editor SQL.
  3. Desative a confirmacao obrigatoria por email em Auth para liberar o fluxo por usuario e senha.
  4. Publique o repositorio no GitHub Pages com a URL https://JonasMMoreira.github.io/gamehub/.
  5. Importe um JSON do Playnite para popular ranking, horas e jogos competitivos.

Repositorio

JonasMMoreira/gamehub Use a branch principal com Pages apontando para a raiz do projeto.

URL final

JonasMMoreira.github.io/gamehub Os assets ja usam caminhos relativos, entao o subdiretorio funciona sem ajustes extras.
